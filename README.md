💖 Hearts to Hearts | Premium Personal Message Engine
Hearts to Hearts adalah platform ekspresi personal kasta tertinggi yang menggabungkan kemewahan visual interaktif ala enterprise agency dengan kedalaman emosional pesan heart-to-heart.

Bukan sekadar pengirim pesan biasa, Hearts to Hearts mengkalibrasi ulang cara kita menyampaikan janji, komitmen, dan perasaan, menjadikannya sebuah cinematic experience yang tak terlupakan bagi penerima.

🚀 Fitur Utama (Enterprise Grade)
💎 Engagement Module (Anti-Tolak Engine V13)
Dirancang khusus untuk momen krusial seperti menyatakan cinta atau melamar. Dilengkapi dengan Enhanced Dodge-Response Technology, di mana tombol "Enggak" akan menghindar secara dinamis dan agresif saat didekati kursor, memastikan hasil interaksi yang positif secara sistem.

🖋️ Vows Archive (Luxury 3D Keepsake)
Fokus pada estetika dan kenyamanan membaca untuk pesan panjang (surat cinta, anniversary, atau apresiasi). Menampilkan animasi True-Center 3D Envelope yang halus, di mana surat muncul tepat di tengah fokus mata penerima, memberikan impresi eksklusif ala boutique agency.

📱 Adaptive Layout Detection
Sistem secara otomatis mendeteksi perangkat penerima (Mobile vs Desktop).

Mobile Optimized: Kertas surat menyesuaikan lebar layar ponsel agar teks tidak terpotong dan nyaman dibaca.

Desktop Interface: Menampilkan kemewahan visual penuh dengan ruang baca yang elegan.

🔗 Secure Private Link & Personalized Alias
Menghasilkan URL unik yang privat. User dapat membuat alias custom (misal: hearts.me/buat-annisa) untuk kesan yang lebih personal dan profesional.

📤 Modern Social Sharing
Menggunakan Web Share API terbaru. Pasangan lu bisa langsung membagikan momen bahagia mereka ke platform media sosial apa pun yang terpasang di HP mereka (WhatsApp, Instagram, etc.) tanpa perlu menyalin nomor terlebih dahulu.

🛠️ Stack Teknologi
Frontend: HTML5, CSS3 (Custom Glassmorphism), Tailwind CSS

Animations: GSAP (GreenSock Animation Platform) untuk transisi super halus, AOS.js

Interactivity: JavaScript (Native), Web Share API, Canvas-Confetti

Deployment: Vercel

💻 Cara Menggunakan
Buka website Hearts to Hearts.

Pilih Mode: Engagement atau Vows Letter.

Di Editor, upload foto terbaik si dia (potrait disarankan).

Tuliskan pesan terdalam lu.

Isi Alias URL Kustom lu.

Klik Deploy Session dan kirim link premium-nya ke dia.
